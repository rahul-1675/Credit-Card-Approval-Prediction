Developer REST API

Integrate CredAI Pulse's credit decisioning engine into external banking systems, CRM portals, or customer checkout funnels.

Endpoint

POST /api/predict

JSON Request Payload Schema

Send a JSON object with the following fields:

Field Name Type Range / Values
AgeInteger18 - 75
GenderString"Male", "Female"
Marital_StatusString"Single", "Married", "Divorced", "Widowed"
Education_LevelString"High School", "Graduate", "Postgraduate", "Doctorate"
Employment_TypeString"Salaried", "Self-Employed", "Retired", "Unemployed"
Employment_Duration_YearsFloat0.0 - 40.0
Annual_IncomeFloat5000.0 - 450000.0
Loan_AmountFloat1000.0 - 200000.0
Existing_DebtFloat0.0 - 150000.0
Credit_ScoreInteger300 - 850 (FICO)
Credit_HistoryInteger1 (No defaults), 0 (Has defaults)
Payment_History_StatusString"On-time", "Delayed"
Credit_InquiriesInteger0 - 10 (Last 6 months)

Sample curl Command

curl -X POST http://127.0.0.1:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{
    "Age": 30,
    "Gender": "Female",
    "Marital_Status": "Married",
    "Education_Level": "Graduate",
    "Employment_Type": "Salaried",
    "Employment_Duration_Years": 4.5,
    "Annual_Income": 95000.0,
    "Loan_Amount": 20000.0,
    "Existing_Debt": 12000.0,
    "Credit_Score": 730,
    "Credit_History": 1,
    "Payment_History_Status": "On-time",
    "Credit_Inquiries": 1
  }'
